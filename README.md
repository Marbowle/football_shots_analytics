# Football Shots Analytics - analiza strzałów oraz pozycji zawodnika przy wykonywnaiu strzału

## Cel projektu 
Celem tego projektu jest stworzenie takiego systemu, który będzie analizował mecze piłkarskie, rozpoznając zawodnika, który wykonuje strzał, oblicza dystan z któego oddawany jest strzał oraz kąt, a następnie oceni, czy pozycja zawodnika sprzyja w zdobyciu gola. 

Projekt jest częścią pracy dyplomowej z zakres analizy sporotwej i uczenia maszynowego.

Materiały wideo używane w tej analizie pochodzą z publicznych źródeł takich jak YouTube oraz SoccerNet. Mecze video ze źródła SoccerNet zostały uzyskane po wypełnieniu NBA, zastrzegającego niekomercyjne użycie pobranych danych.
